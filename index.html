Description Example
A page container with left/right margins. A page container with a fixed-width content panel with left and right margins.
Horizontal field(s) A container for a horizontal field (where the field's label is to the left of the field). If there is more than one field they are stacked vertically. There is also a place for an optional error message.
Vertical field(s) A container for a verical field (where the field's label is above the field). If there is more than one field they are stacked vertically. There is also a place for an optional error message.
A one or two column container of horizontal fields. A column container that shows one and two columns of horizontal fields.
A one or two column container of vertical fields. A column container that shows one and two columns of vertical fields. Note how this example and the 'h-column-container.html' example both use the same HTML (except for CSS class names).
A tabbed UI A tabbed UI (tabs.css) wrapped in a 'page-container'.
